# Creaptica

Aplicación web full-stack moderna con arquitectura separada entre frontend y backend.

## 🎯 Características

- Arquitectura completa de aplicación web (Frontend + Backend)
- Sistema CIIS integrado
- Interfaz moderna con diseño responsivo
- API backend robusta
- Frontend interactivo y dinámico

## 🚀 Tecnologías Utilizadas

### Backend
- **Python (39.5%)** - Lógica de servidor y APIs

### Frontend
- **TypeScript (29.0%)** - Desarrollo tipado y seguro
- **SCSS (19.6%)** - Estilos avanzados y mantenibles
- **HTML (11.9%)** - Estructura semántica

## 📁 Estructura del Proyecto

```
Creaptica/
├── ciis_backend/              # Backend en Python
│   └── [módulos del servidor]
├── ciis_frontend/             # Frontend con TypeScript
│   ├── src/
│   ├── styles/              # Archivos SCSS
│   └── [componentes]
└── .idea/                     # Configuración del IDE
```

## 💻 Instalación

### Backend
```bash
cd ciis_backend
pip install -r requirements.txt
python manage.py runserver
```

### Frontend
```bash
cd ciis_frontend
npm install
npm run dev
```

## 📝 Uso

1. Inicia el servidor backend
2. En otra terminal, inicia el frontend
3. Accede a la aplicación desde el navegador
